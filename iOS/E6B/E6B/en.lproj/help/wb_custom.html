Customize Aircraft

You can customize the aircraft used in weight and balance calculations.

Opening the aircraft editor

In the Weight and Balance list of calculations, click the "Aircraft Editor" in the lower right. An Aircraft Editor will appear allowing you to create and duplicate templates and edit them.

Duplicate an existing aircraft

In the Aircraft Editor, press "Copy Template". This will allow you to pick a pre-defined aircraft and make a copy that can be modified.

Add a new template

You can create a new template from scratch by pressing the '+' button in the lower left.

Duplicate or delete existing templates

Press the 'Edit' button in the upper right. To duplicate, select the aircraft template you wish to duplicate and press the 'Duplicate' button. To delete pick the red delete button on the left of the template and confirm.

Editing a template

You can edit a template by picking the specified aircraft in the template list.

The parameters you need to supply for each aircraft are:

  • Manufacturer The manufacturer who built the aircraft. (i.e., Cessna, Bonanza, Mooney, etc.)
  • Aircraft Type The model or type of aircraft. (i.e., C172R, G36, etc.)
  • Name The name of the template; typically the aircraft's N number.

These parameters are used to help identify the aircraft, and can be anything you wish. The following parameters are used to define the units that are used when specifying the aircraft:

  • Weight The unit used for weight (lbs, kg, etc)
  • Arm The unit used for arm (inches, cm)
  • Moment Unit used for moment (foot-inches, kg-cm).
  • Speed Unit used for speed (knots, mph, etc).

Aircraft Data is data associated with the aircraft as a whole:

  • Va The maneuvering speed for your aircraft
  • Max Weight The maximum weight for your aircraft (used in calculating Va)
  • Empty Weight The measured empty weight of your aircraft.
  • Emtpy Arm The measured empty arm for your aircraft.

Station Locations allow you to define certain predefined arm locations, such as the arm of the pilot's seat or the baggage compartment.

  • Station Data: Name The label used to describe the location (i.e., "Pilot Seat", "Baggage Compartment")
  • Station Data: Arm The distance from the datum for the location

Fuel Tank Locations allow you to define one or more fuel tanks, the carrying capacity of each tank and the type of fuel used. While the system permits an unlimited number of fuel tanks to be specified, the E6B's internal algorithms makes the practical limit around 8 to 10 different station locations for fuel.

  • Fuel Tank: Name A descriptive label for the fuel tank station (i.e., "Main Tanks", "Tip Tanks")
  • Fuel Tank: Volume The maximum carrying capacity of the fuel tank.
  • Fuel Tank: Units The unit used in describing the volume of fuel. (Gallons, Liters)
  • Fuel Tank: Type The type of fuel, typically aviation gas.
  • Fuel Tank: Arm The distance from the datum for this tank.

Weight/Balance Envelope

The data entered here allows you to define the limits of the weight and balance for this aircraft, for different types of aircraft use. For example, you can define the weight and balance limits for normal use and for utility use for a Cessna 172 by entering two weight/balance envelopes. Each graph is specified by a name and by the weight/arm points that define the enclosing polygon for the graph. Units are specified in the weight and arm units above.

  • Envelope Name A descriptive label for the polygon defining the weight and balance limits for this usage. (i.e., "Normal", "Utility")
  • Envelope Data: Weight One of several points for each polygon specifying the weight at that point.
  • Envelope Data: Arm One of several points for each polygon specifying the arm at that point.
