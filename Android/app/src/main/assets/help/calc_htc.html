Heading for True Course

This calculation will compute the true heading and ground speed given your desired true course and true airspeed, and the wind direction and speed. This calculation can be used in flight planning to determine the necessary true heading to maintain a ground true course between two points.

Sample Problem

Wind direction and speed is 120° at 15 kt. True course is 80°, true airspeed is 110 kt.

Answers:

Entering the data you discover:

True Heading 85.03
Ground Speed 98.09 kt
